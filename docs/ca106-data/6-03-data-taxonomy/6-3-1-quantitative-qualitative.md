## 6.3.1 Data categories

---

## Definition and Purpose
Data is categorised to describe its **nature and characteristics**.  
Understanding data categories supports appropriate selection of **storage, processing, and analysis methods** in digital software development.

---

## Quantitative Data
- **Quantitative data** represents numerical values
- It can be measured or counted
- Used where values can be expressed as numbers

**Characteristics:**
- numerical
- suitable for calculations and statistical analysis
- commonly structured

---

## Qualitative Data
- **Qualitative data** represents non-numerical values
- Describes qualities, characteristics, or attributes

**Characteristics:**
- descriptive
- not inherently numerical
- commonly unstructured

---

## Purpose of Data Categorisation
- Supports appropriate data handling
- Influences:
  - storage methods
  - processing techniques
  - analysis approaches

Incorrect categorisation can reduce data usefulness.

---

## Exam Focus
You should be able to:
- Define quantitative and qualitative data
- Distinguish between the two categories
- Identify the appropriate category for a given data example

!!! tip "Exam tip"
Marks are often awarded for **correctly categorising data**, even when no calculations are required.
