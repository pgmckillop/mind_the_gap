## 6.3.2 Structured data

---

## Definition and Purpose
**Structured data** is data that is organised into a predefined format, making it easy to store, search, and analyse.  
Structured data is primarily used to represent **quantitative data** in digital software development.

---

## Characteristics of Structured Data
- Organised into rows and columns
- Follows a defined schema or structure
- Data items are easily identifiable

**Common characteristics include:**
- predictable format
- efficient querying
- suitable for automated processing

---

## Relationship to Quantitative Data
- Quantitative data is **structured**
- Numerical values fit naturally into fixed formats
- Enables efficient calculations and comparisons

---

## Purpose of Using Structured Data
- Supports efficient data storage
- Enables fast searching and retrieval
- Simplifies data analysis and processing

---

## Exam Focus
You should be able to:
- Define structured data
- Explain its purpose
- Recognise that quantitative data is structured

!!! tip "Exam tip"
Questions may test understanding that **structure supports efficiency**, not just storage.
