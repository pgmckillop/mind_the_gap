## 6.3.3 Unstructured data

---

## Definition and Purpose
**Unstructured data** is data that does not follow a predefined format or schema.  
It is primarily used to represent **qualitative data** in digital software development.

---

## Characteristics of Unstructured Data
- Not organised into fixed rows and columns
- Does not follow a rigid structure
- Often stored as a single item

**Common characteristics include:**
- variable format
- difficult to search directly
- requires processing before analysis

---

## Relationship to Qualitative Data
- Qualitative data is **unstructured**
- Descriptive data does not naturally fit into fixed formats
- Often requires transformation before structured use

---

## Purpose of Using Unstructured Data
- Captures rich, descriptive information
- Supports flexibility in data collection
- Enables storage of complex or varied content

---

## Exam Focus
You should be able to:
- Define unstructured data
- Explain its purpose
- Recognise that qualitative data is unstructured

!!! tip "Exam tip"
Marks are awarded for correctly linking **qualitative data** with **unstructured storage**, not just defining the terms.
